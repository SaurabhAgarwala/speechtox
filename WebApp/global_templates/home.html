ABOUT

SpeechTox is a WebApp made using HTML, CSS, JS and Django which integrates a Machine Learning model.

The machine learning model has been trained on the following Kaggle dataset: https://www.kaggle.com/c/jigsaw-toxic-comment-classification-challenge. It is used to predict the following categories given any text: toxic, severe_toxic, obscene, threat, insult and identity_hate. The model is trained using Logistic Regression on the text after tokenising it using tf-idf.

The WebApp uses this model for three applications:
1)Given any text, it lists the categories that it fits.
2)Given a song title and artist, it again displays the categories it fits.
3)Given a billboard playlist and number of items, it classifies the songs in the playlist into two categories: decent and indecent.
An additional feature that we will add is to display the censored lyrics of any song entered by the user.
